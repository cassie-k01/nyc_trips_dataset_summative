About the Project

The NYC Taxi Trip Dashboard is an enterprise-level fullstack application that visualizes patterns in New York City taxi trip data. It enables users to analyze pickup/dropoff trends, fare distributions, and driver performance through an interactive web interface.

Dataset Information

Data Source: NYC Taxi and Limousine Commission (TLC) Dataset. This dataset contains trip-level records including timestamps, distances, fares, tips, and geolocations.
